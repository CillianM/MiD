# Web Application
This directory contains all the source code of the admin interface and dummy backend

## Web UI
This is the sample interface that allows one to test the identifying party's side of the workflow

### Dependencies
- [Angular CLI](https://cli.angular.io/)
- [Bootstrap](https://getbootstrap.com/)

## Web Backend
This is the dummy backend that provides the HTTP methods necessary to wrap the UI calls with the correct authentication headers

### Dependencies
- [Spring](https://spring.io/)
- [Swagger](https://swagger.io/)
